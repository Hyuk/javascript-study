# JavaScript Basic

## Comment
* in-line comment
* multi-line comment 

### in-line comment
```javascript
// this is an in-line comment
```

### multi-line comment
```javascript
/* this is a
multi-line comment */
```

## Variables
* number
* string
* symbol
* object
* boolean
* null
* undefined

### number
```javascript
var num1 = 12;
console(num1); // 12
```

### string
```javascript
var text1 = "Hello World";
console(text1); // "Helllo World"
```
